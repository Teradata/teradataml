## Workflows showcasing capabilities of Teradata Python package for Feature Store

teradataml provides following feature store managements:

* ##### Feature Store: Centralized repository for storing, sharing, and managing features for machine learning models.
* ##### Data Domains: Logical grouping of related datasets and features.
* ##### Dataset Catalog: Organized listing of available datasets for feature engineering and model training.
* ##### Feature Process: Workflows for creating, transforming, and managing features.

## Following workflows in the form of Jupyter notebooks are provided:
* #### Enterprise Feature Store Notebooks
    - ##### `EFS-DataDomain.ipynb`: Managing data domains in the feature store.
    - ##### `EFS-DatasetCatalog.ipynb`: Cataloging datasets for feature engineering.
    - ##### `EFS-FeatureProcess.ipynb`: Feature creation and transformation processes.
    - ##### `EFS-FeatureStore.ipynb`: Core feature store operations and examples.
